# Project Brief: Salmen Khelifi Portfolio ## 1. Project Overview A high-fidelity, technical portfolio for Salmen Khelifi, a Full-Stack Software Engineer & Automation Specialist. The project focuses on communicating engineering depth, architectural clarity, and a premium "anti-hype" professional brand. ## 2. Visual Identity: "Refined Dark Architecture" The design system leverages a high-contrast, dark-mode aesthetic inspired by premium engineering tools (Linear, Vercel). * **Primary Palette:** * Base: `#08080a` (Near-black) * Surface: `#0e0e12` / `#16161f` * Accent: `#2f80ed` (Steel Blue) * **Typography:** Manrope (Weights: 300-800). Negative letter-spacing on headlines. * **Materiality:** "Liquid Glass" — 20px backdrop blur, 8-13% white opacity, hairline borders with top-weighted highlights. * **Atmosphere:** Subtle ambient depth using low-opacity blurred radial gradients (Blue, Purple, Teal). ## 3. Core Experience Pillars * **Engineering First:** Using architecture diagrams and system topology flows rather than generic lifestyle imagery. * **Editorial Layout:** Asymmetric grids and generous whitespace to create a calm, high-end feel. * **Inclusive Design:** Full support for `prefers-reduced-motion` and `prefers-reduced-transparency` modes. * **Conversion-Focused:** Integrated "Start a Project" inquiry system designed for technical discovery. ## 4. Technical Requirements * **Performance:** Optimized for Core Web Vitals; minimal layout shift (CLS) during material rendering. * **Accessibility:** WCAG AA compliance, 44px+ touch targets, and clear keyboard focus states. * **Responsive Architecture:** System diagrams must transition from horizontal ecosystems (Desktop) to vertical flows (Mobile). * **Print-Friendly:** A specialized résumé stylesheet that strips ambient effects and preserves semantic hierarchy for PDF/Print output. ## 5. Key Screens & Systems 1. **Homepage:** Hero with ecosystem visual, asymmetric project showcase, and engineering philosophy. 2. **Work/Case Studies:** Technical deep-dives into real projects like "Luxe Spa Booking" and "Anlingo AI." 3. **Inquiry System:** Multi-step technical discovery form for new project requests. 4. **Component Library:** Standardized modules for Featured, Standard, and Compact project displays. 5. **Technical Diagrams:** A reusable node-and-connection system for visualizing backend architecture.
