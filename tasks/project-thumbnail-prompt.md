# Project Thumbnail Generation Prompt

Use this prompt with ChatGPT image generation. Attach one or more real project screenshots first.

```text
Create a high-quality portfolio project thumbnail from the attached project screenshot(s).

Goal:
Make a clean, professional thumbnail for a developer portfolio work section. It should look like a polished case-study preview, similar to a Freelancer.com project thumbnail: sharp, modern, realistic, and based on the real UI in the screenshots.

Requirements:
- Use the actual project UI from the screenshot as the main subject.
- Keep the design faithful to the original project: colors, layout style, brand feel, and product type.
- Improve the presentation quality: better framing, cleaner lighting, sharper contrast, and a premium portfolio look.
- Use a 16:10 or 4:3 landscape composition.
- Show the screen inside a realistic browser window, laptop mockup, desktop monitor, or clean floating UI frame.
- Add a subtle background that matches the project style, but do not overpower the UI.
- Make the UI readable and crisp.
- Avoid fake text-heavy redesigns. Do not invent unrelated app screens.
- Do not use random stock photos unless the project itself is product/fashion/real-estate/food based.
- Do not add logos or brand names that are not in the screenshot.
- Do not include watermarks.

Output:
- One polished thumbnail image.
- Landscape format.
- Suitable for a portfolio grid/card.
- High resolution, at least 1400px wide.

Style direction:
Premium SaaS/product case-study thumbnail, clean shadows, soft depth, realistic screen presentation, modern tech portfolio quality.
```

Optional project-specific line to add:

```text
This project is about: [short project description]. Emphasize: [dashboard / e-commerce / mobile app / admin panel / automation workflow / blog / fashion store].
```
